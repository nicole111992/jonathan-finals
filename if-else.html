About

If else statement is a condition statement that is used in the execution of a computer program in pre-defined rules. The if-else statement helps you to run a specific block of a program if the condition is true or else, it will check other conditions.

You already know that Java supports the usual logical conditions from mathematics:

Less than: a < b

Less than or equal to: a <= b

Greater than: a > b

Greater than or equal to: a >= b

Equal to a == b

Not Equal to: a != b

Syntax

if(condition) { // Statements inside body of if }

else { //Statements inside body of else }
